# Flask Devops Project

## Getting started

Set up virtual env:

```bash
python3 -m venv .venv
./venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the application:

```bash
flask run --host=0.0.0.0 --port=5000 --debug
```

Access the applications: <http://localhost:5000>
